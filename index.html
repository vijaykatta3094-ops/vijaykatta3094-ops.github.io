Local Brain

Continuous evaluation pipeline using a 14B-parameter LLM running entirely on local GPU. Scores my own work daily, extracts structured signals via constrained generation (skill gaps, preferences, project state), and writes them back into a memory store that the rest of the stack reads. A self-supervised data flywheel — $0 inference cost, zero data leaving the house.

Sub-second voice-to-text deployed on consumer GPU. Custom audio capture pipeline that handles 30+ failure modes the popular libraries don't (Bluetooth SCO dropouts, mic source switching, silence hallucinations), Nvidia Parakeet TDT for transcription, OS-level paste-anywhere integration. Production-grade — I use it every day. Public on GitHub.

perf-engineering · gpu

GPU Coach

Performance-engineering tool for local LLM inference. Captures GPU telemetry, applies roofline analysis to classify the active bottleneck regime (compute-bound, bandwidth-bound, VRAM-capped, thermal-throttled, PCIe-bottlenecked, or idle), then ranks concrete optimizations that exploit the unused dimension. Calibrated against RTX 5090 + Qwen3-14B.

ci · dev-tools

/validate

Auto-validation dispatcher for AI-assisted development. Inspects the git diff to decide which categories apply (frontend, backend, ML, skill), runs the right validator per category, and returns one PASS/FAIL line with evidence — no rubber-stamping. Found six real production bugs the first time it ran on a real codebase, including a misclassified systemd directive and broken YAML frontmatter across multiple skills.
